# 🧠 About EDM Lab

**EDM Lab** is a research-driven organization focused on developing intelligent systems that improve education through **data-driven insights**.

Our mission is to:

- 📊 Analyze learning behavior through **Educational Data Mining**
- 🤖 Build **AI-powered learning systems**
- 📈 Develop **predictive models for student success**
- 🌐 Create **open-source tools for research and education**

---

# 🔬 Research Areas

| Area | Description |
|-----|-------------|
| 📚 Educational Data Mining | Mining patterns from student learning data |
| 🤖 Artificial Intelligence | Intelligent tutoring systems and AI education tools |
| 📊 Learning Analytics | Data-driven insights for learning improvement |
| 🧮 Machine Learning | Predictive models for academic performance |
| 🧑‍💻 Software Engineering | Platforms and systems for learning environments |
